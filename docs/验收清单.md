## 七、0.5 专项验收（稳定性与长停顿）

| 编号 | 验收项 | 步骤 | 通过标准 | 优先级 |
|---|---|---|---|---|
| B-01 | 长停顿完整上屏 | 按住说话 10 秒，中间停顿 2-3 秒，再继续说，松键后观察上屏 | 30 次重复测试，文本完整率 100% | P0 |
| B-02 | 收尾阶段可见 | 松键后观察状态文案 | 明确显示 `收尾中`，完成后回 `IDLE` | P1 |
| B-03 | Final 缺失兜底 | 人为注入 Final 丢失场景 | 仍可上屏 partial 回退文本，且有日志记录 | P0 |
| B-04 | 自动软重连 | 识别链路超时或断流 | 自动软重连成功并恢复识别，无需手动重启 | P0 |
| B-05 | 自动硬重置 | 连续软重连失败后继续触发 | 进入硬重置并在 10 秒内恢复到可识别状态 | P0 |
| B-06 | 长时间连续使用 | 执行 200 次短按 + 50 次长按 | 会话成功率 >= 99%，无“必须重启应用”案例 | P0 |
| B-07 | 失败可诊断 | 制造网络异常/鉴权异常 | 日志包含 `session_id`、`state`、`error_code`、最近事件快照 | P1 |
| B-08 | 状态可感知 | 观察托盘与悬浮窗文案 | `录音中/收尾中/重连中/失败可重试` 与真实状态一致 | P1 |

## 八、0.6 专项验收（会话一致性与体验）

| 编号 | 验收项 | 步骤 | 通过标准 | 优先级 |
|---|---|---|---|---|
| C-01 | 会话令牌隔离 | 模拟旧会话迟到 Final 事件 | 非当前会话事件被丢弃，且有 `stale_event_drop_count` 记录 | P0 |
| C-02 | 一次会话一次完整上屏 | 长句 + 停顿 + 继续，松键后观察输出 | 不出现“本次半段 + 下次补尾段”问题，100 次通过率 >= 99% | P0 |
| C-03 | 字幕连续性 | 说话中停顿 2-3 秒再继续 | 字幕历史内容不清空，思路不断流 | P0 |
| C-04 | 焦点可编辑提示 | 在输入框/桌面/不可编辑区域切换 | Overlay 与托盘提示准确反映 `editable/unknown/not_editable` | P1 |
| C-05 | 快速收尾路径 | 短句快速输入后松键 | P95 `finalize_latency_ms` < 1500ms，P99 < 2500ms | P0 |
| C-06 | 菜单中文化 | 打开托盘菜单检查文案 | 菜单与关键状态文案中文覆盖率 100% | P1 |
| C-07 | 0.6 诊断字段完整 | 导出诊断快照并检查 JSON | 包含 `session_token`、`stale_event_drop_count`、`fast_finalize_hit_count` 等字段 | P1 |
| C-08 | 收尾稳定性回归 | 连续 50 次停顿场景输入 | 无卡死、无重复粘贴、无跨会话串流 | P0 |
