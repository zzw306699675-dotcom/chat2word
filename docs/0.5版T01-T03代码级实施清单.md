## 1. 目标与完成标准

目标：
1. 松键后不再“立即结束会话”，而是进入收尾阶段。
2. 长停顿场景中，尾段文本能被正确接收并上屏。
3. 收尾逻辑具备明确超时、错误和日志语义。

完成标准（DoD）：
1. 新状态路径可达：`IDLE -> RECORDING -> FINALIZING_GRACE -> RECOGNIZING_DRAIN -> PASTING -> IDLE`。
2. `stop_session()` 幂等且不会导致资源泄漏。
3. 新增/修改测试全部通过，且覆盖新分支。

     - `_sentinel_emitted: bool`
